# 30-specs

Precise, testable component specifications. A spec here is the **prerequisite
for any code** ([ADR-0001](../20-decisions/adr-0001-documentation-first-development.md)).
New specs start from [`../50-meta/templates/spec.md`](../50-meta/templates/spec.md)
and are created per component immediately before it is built (Stage 3).
