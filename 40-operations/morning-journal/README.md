# Morning Journal

Daily usage debriefs of the living phase ([ADR-0020](../../20-decisions/adr-0020-living-phase-trust-before-intelligence.md)).
One file per real morning: `YYYY-MM-DD.md`. Alex's answers may be in German;
structure stays English.

The journal is the evidence base for the next roadmap decision. "Nothing to
report" is a valid entry. Trust-invariant candidates spotted here go to
[trust-invariants.md](../../00-product/trust-invariants.md).

## Entry template

```markdown
# Morning — YYYY-MM-DD

- Sources live: calendar yes/no · email yes/no · model real/mock
- Opened AIOS before Outlook: yes/no
- Still felt the pull to open Outlook immediately: yes/no — why?

## Genuinely useful
## Unnecessary / too much
## Missing / too little
## Surprising
## Missed something important?
## A recommendation I actually trusted?
## Cognitive load after 30 seconds: lower / same / higher — why?
## Corrections given (verbs used)
## Bug / usability notes (freeze-compliant fixes only)
## Trust-invariant candidate?
```
